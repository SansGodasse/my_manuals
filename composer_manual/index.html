Mon super manuel de composer

Qu'est-ce que c'est que ce truc ?

Ce truc permet d'installer des librairies facilement sans avoir à bidouiller à la main.

Utilisation

composer init A exécuter dans le dossier du projet pour créer le fichier composer.json de manière intéractive.
composer install Installe les packages nommés dans le fichier composer.json.
composer update Met à jour le dossier vendor si on a fait des modifs sur composer.json.

Fichier composer.json

Voici un exemple de fichier composer.json pour comprendre la structure :

{
    "name": "nom_de_l_auteur/nom_du_projet",
    "description": "La description du projet (facultatif)",
    "require": {
        "auteur_de_la_librairie/nom_de_la_librairie": "version",
        "michelf/php-markdown": "^1.8"
    },
    "authors": [
        {
            "name": "nom_de_l_auteur",
            "email": "email_de_l_auteur@gmail.com"
        }
    ],
    "autoload": {
        "psr-4": {"nom_du_namespace\\": "mes_librairies_perso/MesClasses"}
    }
}
